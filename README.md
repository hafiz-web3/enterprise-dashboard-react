# Enterprise Dashboard (React + TypeScript)

Enterprise-style dashboard built to demonstrate frontend architecture and UI patterns commonly used in internal tools, operational dashboards, and management portals.

## What this project demonstrates
- Modular React component architecture
- Type-safe development with TypeScript
- KPI-driven dashboard layouts
- Data tables and operational views
- Clean separation of data, components, and pages

## Tech stack
- React
- TypeScript
- Mocked JSON data (no backend)

## Notes
This project intentionally uses mock data and simplified UI to demonstrate structure and engineering approach without exposing proprietary or client-specific logic.
